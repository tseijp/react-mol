# Introduction

Let's discover **react-mol in less than 5 minutes**.

## Installation

To install the entire `react-mol` lib:

```shell
yarn add react-mol
```

## Getting Started

```shell
git clone https://github.com/tseijp/react-mol
```

Run the development server:

```shell
cd react-mol
yarn init
yarn docs
```

Your site starts at [http://localhost:3000](http://localhost:3000).
