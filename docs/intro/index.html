Introduction

Let's discover react-mol in less than 5 minutes.

Installation#

To install the entire react-mol lib:

yarn add react-mol

Getting Started#

git clone https://github.com/tseijp/react-mol

Run the development server:

cd react-mol
yarn init
yarn docs

Your site starts at http://localhost:3000.
